## 📌 Overview

**Kreeda-Prerana-Scout** is an internship project focused on building a modern and intelligent sports scouting application. The platform is designed to support athlete discovery, scouting workflows, and better evaluation processes through a clean user experience and AI-powered capabilities.

The repository is currently private and maintained by **Nithin P** under the GitHub account **Nithi-121**. It is built primarily with **TypeScript** and is generated from the **Google Gemini AI Studio repository template**.

## 🚀 Getting Started

### Prerequisites

Before running the project locally, ensure that **Node.js** is installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Nithi-121/Kreeda-Prerana-Scout.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Kreeda-Prerana-Scout
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Create a `.env.local` file and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```
